# Feedback Collection

At each skill's end, invite a one-line PM reaction and persist it.

- Ask: "Anything off, or want to change anything before we move on?"
- Write to `pipeline/{slug}/eval/feedback.md` tagged with the skill name and phase.
- Feedback is signal for `/eval` Layer 3 (PM conviction vs. auto-scored rigor). Keep it lightweight — one prompt, not a survey.
